# 🤖 Gemini AI Studio Toolkit

A comprehensive **Google Gemini AI** toolkit with all major AI features — built in Google AI Studio style.

## ✨ Features

| Feature | Model | Description |
|---------|-------|-------------|
| 🎙️ Audio Transcribe | gemini-2.0-flash | Real-time audio to text |
| 📹 Video Analyze | gemini-2.5-pro | Video content analysis |
| ⚡ Fast Response | gemini-2.0-flash-lite | Low-latency responses |
| 🔍 Image Analyze | gemini-2.5-pro | Document & image analysis |
| 🖼️ Image Generate | imagen-3.0 | High-quality image generation |
| 🎬 Video Generate | veo-3 | Text to video |
| 🤖 AI Chatbot | gemini-2.0-flash | Multi-turn conversation |
| 🔍 Google Search | gemini + search | Real-time search grounding |
| 🔊 Text to Speech | gemini-tts | Natural voice synthesis |
| 🎵 AI Music | lyria-2 | Music generation |

## 🚀 Quick Start

### 1. Get Your Gemini API Key
Go to [Google AI Studio](https://aistudio.google.com/apikey) and create a free API key.

### 2. Open the App
Simply open `index.html` in your browser — no server needed!

### 3. Add Your API Key
Go to **⚙️ API Settings** and paste your Gemini API key.
